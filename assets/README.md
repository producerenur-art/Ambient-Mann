# Bilder (assets/)

Legg bildene dine i denne mappa med **nøyaktig disse filnavnene**. Siden peker
allerede på dem — mangler et bilde, vises bare alt-tekst (ingen krasj).

| Filnavn | Hva | Vises som |
|---------|-----|-----------|
| `universe-bg.jpg` | Stjernehimmelen (universet) | Bakgrunn (bevegelige stjerner oppå) |
| `ambient-mann-logo.png` | «AM – Ambient Mann»-logoen (helst transparent PNG) | Hovedlogo i toppen (m/ bevegelige skyer) |
| `ambient-mann-portrait.jpg` | Portrettet ditt (silhuetten) | Bunn-logo → radioq37.com/artist/ambient-mann |
| `feedfreq-logo.png` | FeedFreq-logoen | Bunn-logo → FeedFreq-gruppa |
| `new-message-from-god.jpg` | «The New Message From God»-kunsten | Bunn-logo |
| `psybient-logo.png` | psybient.org-logoen | Bunn-logo → psybient.org |
| `diceradio-logo.png` | Dice Radio-logoen | Bunn-logo → diceradio.gr |
| `trancentral-logo.png` | Trancentral-logoen | Bunn-logo → trancentral.tv |
| `it-athens-logo.png` | IT Athens (Wake the Beat) | Bunn-logo → ra.co/clubs/212119 |
| `newmessage-logo.png` | The New Message From God (diamant-logo) | Bunn-logo → newmessage.org |
| `siriusfm-logo.png` | Sirius FM (jordkloden) | Bunn-logo → siriusfm.no |

Valgfritt:
- `booking.png` — eget booking-bilde (ellers vises en booking-knapp).
- `vipps-qr.png` — Vipps-QR i donasjons-boksen.
- `icon-192.png`, `icon-512.png` — egne PWA-app-ikoner.

Alle stier og lenker styres i `js/config.js` (`assets` + `links`), så du kan
endre filnavn/lenker der hvis du vil.
